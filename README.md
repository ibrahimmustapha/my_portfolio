You get to see all my projects here🎉

## Prerequisite
- install nodejs (latest version)
- install npm
- internet access 

## How to use 
- copy repo url from github
- run ``git clone codeurl`` in the terminal 
- run ``npm install``
- run ``npm start`` to run the code
- done ~ Happy Coding🎉🎉
